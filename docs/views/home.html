Challenges

  • Dealing with submodules in github

  • Microservice UI composition

  • PWA in microservices

  • Nginx - Even a slash at the URI end matters!

  • Creating associations with sequelize and querying them

  • Keeping balance between coding and reading














Lessons Learned

  • CI/CD!!! It is a BIG part of SDLC and needs time to do it right

  • Sometimes no matter how fast you want to go, things move with their own pace - i n c h by i n c h, therefore, chill!

  • Docker cache!!! It can cause all sorts of weird behavior. Make sure you clean it time to time

  • Keeping boilerplates ready was helpful














Improvements

  • Encrypt communication between all microservices

  • Increase PWA coverage

  • Add more charts for analysis

  • Integrate more log source types to search the IOCs against (e.g Splunk, DB)

  • Build more robust CI/CD pipeline
